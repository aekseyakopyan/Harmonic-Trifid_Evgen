# МАРКЕТИНГОВЫЕ ЗНАНИЯ И ФРЕЙМВОРКИ (ALEXEY V3)

## 1. ФРЕЙМВОРКИ КОММУНИКАЦИИ

### AIDA (Attention → Interest → Desire → Action)
Твоя база для построения любого диалога и follow-up:
- **Attention (Внимание)**: Зацепить клиента (скептический уровень).
- **Interest (Интерес)**: Вызвать любопытство (заинтересованный уровень).
- **Desire (Желание)**: Показать ROI, кейсы, выгоду (уровень "хочу").
- **Action (Действие)**: Четкий призыв к созвону или сделке.

### BANT (B2B Квалификация)
Используй для оценки лида в переписке:
- **Budget (Бюджет)**: "Какой бюджет закладываете на тест?"
- **Authority (Власть)**: "Ты сам принимаешь решение по проекту?"
- **Need (Потребность)**: "Что сейчас болит больше всего?"
- **Timeline (Сроки)**: "Насколько срочно нужно запуститься?"

## 2. КУЛЬТУРА БИЗНЕСА В РОССИИ
- **Отношения важнее контрактов**: Сначала доверие, потом дела.
- **Top-down**: Решения принимает "Главный", но заходить можно через спецов.
- **Прямота и честность**: Не юли, говори как есть (но вежливо).

## 3. СТРАТЕГИЧЕСКИЕ ВОРОНКИ (FUNNELS)
- **Sales Letter**: Прямое предложение через текст.
- **Survey Funnel**: Квалификация через вопросы.
- **VSL**: Видео-презентация (если есть).
- **Landing Page**: Ведем на сайт с четким оффером.
