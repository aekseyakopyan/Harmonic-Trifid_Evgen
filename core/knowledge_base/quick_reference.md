# Шпаргалка менеджера: Быстрая справка по B2B продажам в мессенджерах

**Формат:** Краткая справка для печати или закрепления на стол  
**Размер:** 1 лист А4 можно распечатать и ламинировать

---

## ПЕРВОЕ СООБЩЕНИЕ — ФОРМУЛА

```
[Имя] + [Источник] + [Боль] + [Результат] + [Вопрос] + [Стоп]
```

**Telegram (50-100 слов):**
```
Привет, [ИМЕЯ]. Нашел тебя в [ИСТОЧНИК].

Вижу, что [БОЛЬ]. Помогли похожим [РЕЗУЛЬТАТ].

Обсудим?

P.S. стоп → не буду писать
```

**WhatsApp (100-150 слов):**
```
Здравствуйте, [ИМЯ].

Я [ТВОЕ ИМЯ] из [КОМПАНИЯ]. Помогаем [ЦЕЛЕВАЯ] 
решить [БОЛЬ].

Недавно [РЕЗУЛЬТАТ] для похожей компании.

Было бы интересно обсудить вашу ситуацию?
15 минут разговора.

Если неактуально → просто не отвечайте.
```

---

## НА ВОПРОС "СКОЛЬКО СТОИТ?" — АЛГОРИТМ

| Шаг | Действие | Пример |
|-----|----------|--------|
| 1 | НЕ отвечай ценой! | "Сориентирую..." |
| 2 | Уточни потребность | "Что важнее: А или Б?" |
| 3 | Предложи выбор ИЗ 2 | Вариант 1 и Вариант 2 |
| 4 | Аргументируй разницу | "Первый быстрее, второй дешевле" |
| 5 | Закрой действие | "Договорились?" |

---

## ВОЗРАЖЕНИЕ "ДОРОГО" — 4 ШАГА

1. **Выслушай** → "Понимаю"
2. **Уточни** → "В пределах какой суммы?"
3. **Аргументируй** → объясни разницу, не защищай цену
4. **Закрой** → "Подходит?"

**ГЛАВНОЕ:** Не защищайся, объясни выгоду.

---

## СКОРОСТЬ ОТВЕТА

| Время | Результат |
|-------|-----------|
| 5-15 мин | ✓ ОТЛИЧНО |
| 15-60 мин | ✓ Хорошо |
| >1 часа | ✗ Клиент к конкурентам |

**Если не можешь ответить → отправь автоответ с датой.**

---

## МАКСИМУМ 2 ВАРИАНТА

❌ "Есть еще 5 опций"  
✓ "Вариант А (дешевле) или Б (полнее)?"

Выбор ИЗ 2 = 40% вероятность согласия  
Выбор ИЗ 5+ = паралич анализа

---

## КАЖДОЕ СООБЩЕНИЕ = ВОПРОС

❌ "Вот наш прайс. Как вам?"  
✓ "Какой вариант больше подходит: А или Б?"

Вопрос = форс действия, инициатива остается у тебя.

---

## ИНТЕРВАЛЫ МЕЖДУ КАСАНИЯМИ

- **После 1го письма** → 3-5 дней
- **Follow-up цепочка** → 3-4 дня между письмами
- **Максимум касаний** → 3-4 (потом 6 месяцев пауза)
- **После "подумаю"** → обязательно дата: "напишу завтра в 10:00"

---

## ВРЕМЕНА ДЛЯ ОТПРАВКИ

**ИДЕАЛЬНО:**  
Вт-Чт, 10:00-16:00 по часовому поясу клиента

**ИЗБЕГАЙ:**  
- Пн (планы), Пт (спешка)  
- Выходные, праздники  
- Ночь (сообщение потеряется)

| ДА | НЕТ |

---

## ОШИБКИ И ИСПРАВЛЕНИЯ

| Ошибка | Исправление |
|--------|------------|
| Стена текста | Максимум 5-7 строк |
| Сразу цена | Сначала 1-2 вопроса |
| Много вариантов | Максимум 2 |
| Без дедлайнов | "Напишу завтра в 10:00" |
| Ночная отправка | Рабочие часы (10-16) |
| Агрессив. тон | Мягкий, вежливый |
| Без благодарности | "Спасибо за быстрый ответ" |

## КЛЮЧЕВЫЕ ЦИФРЫ (ОРИЕНТИРЫ)

| Метрика | Хороший уровень |
|---------|-----------------|
| Reply rate | 15-25% |
| Время 1го ответа | 5-15 мин |
| Conversion ответ→встреча | 20-40% |

100 сообщений → 15-25 ответов → 3-10 встреч → 1-2 сделки

## БЛАГОДАРНОСТЬ (ПРАВИЛО)

❌ "Спасибо за внимание!"  
✓ "Спасибо, что быстро ответил. Видно, что вы четко 
знаете, что нужно для своего проекта."

Благодарность за КОНКРЕТНОЕ действие = выделяешься.

---

## БЫСТРАЯ ДИАГНОСТИКА

**Проблема:** Низкий reply rate  
→ Улучши персонализацию, конкретность болей

**Проблема:** Долгие ответы на "сколько стоит"  
→ Задай вопросы до ответа на цену

**Проблема:** Теряются клиенты  
→ Ведите CRM/Таблицу, фиксируйте дедлайны

**Проблема:** Клиент уходит к конкурентам  
→ Отвечай быстрее (5-15 мин максимум)

3. **Сомневаешься** → раздел "Типичные ошибки"
4. **Нужен пример** → раздел "Готовые примеры по отраслям"
